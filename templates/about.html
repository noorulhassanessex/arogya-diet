About Arogya

Arogya is a diet & fitness management tool that helps you track meals, hydration, and nutrition goals while providing structured meal and workout plans.

🍽 Meal Tracking 💧 Hydration 📊 Weekly Reports 🏋 Workout Plans

Our Mission

We aim to promote healthy eating habits by helping users log meals, monitor calorie intake, and follow personalised diet plans. Small consistent changes create long-term health benefits.

What You Can Do

  • Track calories and nutrients daily
  • Monitor hydration goals
  • Calculate BMI and get recommendations
  • Follow workout plans based on your goal
  • Generate weekly reports to see progress

Why We Built This Project

This project demonstrates full-stack web development using Flask, HTML and CSS with dynamic data tracking, clean routing, and a modern user interface.

Tech Used

  • Flask (Python)
  • SQLite database
  • HTML / CSS templates
  • Matplotlib (graphs)
  • ReportLab (PDF reports)
